## In ten minutes, you will

1. Create an organization.
2. Connect (or create) a workspace repository.
3. Activate a template workflow.
4. Transition your first issue end-to-end.
5. Watch the change land on-chain.

## What you need

- A GitHub account (so we can create a workspace repo for you — or connect an existing one).
- Nothing else. No credit card, no credentials to configure.
